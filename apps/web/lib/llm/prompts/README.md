# LLM Prompts

Todos os system prompts versionados aqui. **Não inline em código** — facilita iteração, comparação e tracking.

## Estrutura

- `apply-coach.md` — system prompt do Apply Coach (Coach chat conversational)
- `trilha-hidratacao.md` — system prompt do engine de geração de trilhas
- `floating-assistant.md` — versão simplificada do Apply Coach pro FAB
- `classificacao-feedback.md` — Haiku classifica feedback de lição

## Convenção

- Markdown templated com `{{variavel}}`
